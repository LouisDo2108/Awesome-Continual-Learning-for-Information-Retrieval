# Awesome Continual Learning for Information Retrieval

Continual Learning (CL) (also known as Incremental Learning or Lifelong Learning) in Information Retrieval (IR) is gaining increasing attention and calls for a well-curated collection of relevant literature.
This repository aims to serve as a resource for the IR community and related research areas.
