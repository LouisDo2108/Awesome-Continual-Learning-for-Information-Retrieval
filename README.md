# Awesome Continual Learning for Information Retrieval

Continual Learning (CL) (also known as Incremental Learning or Lifelong Learning) in Information Retrieval (IR) is underexplored; however, it is gaining increasing attention which neccessitates a well-curated collection of relevant literature.

This repository aims to serve as a resource for the IR community and related research areas.

## Note
In this repository, we focus exclusively on CL for IR and do not cover CL in other domains. CL has been extensively studied, particularly in the computer vision community.
To avoid redundancy and keep this collection focused, we recommend the following comprehensive survey for a broader overview of CL:

[**TPAMI'24**] A Comprehensive Survey of Continual Learning: Theory, Method and Application. [[Paper](https://arxiv.org/abs/2302.00487)][[Github](https://github.com/lywang3081/Awesome-Continual-Learning)]

We also recommend this curated list: [**xialeiliu/Awesome-Incremental-Learning**](https://github.com/xialeiliu/Awesome-Incremental-Learning), which is comprehensive but does not specifically cover CL for IR.
